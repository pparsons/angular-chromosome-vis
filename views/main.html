Getting Started

Dependencies

  • AngularJS
  • D3
  • underscore
  • JSDAS

Installation

Install with bower: 

bower install angular-chromosome-vis --save

Declare a dependency on the angular module:

angular.module('myModule', ['angularChromosomeVis']);

Settings

All settings can be provided as attributes on the chromosome directive.

  • chr: The chromosome to represent (1-22, X, Y).

  • rel-size: Use relative or absolute sizing (true or false).

  • axis: Show axis under chromosome for coordinates (true or false).

  • assembly: Genome assembly to use (e.g., 37, 38).

  • width: Width of svg element.

  • height: Height of svg element.

  • mode: Selection mode (multi, single, none).

Usage and Examples

Add the directive to HTML:

as an attribute:

<div chromosome chr="1" rel-size="true" axis="true" assembly="37" width='850' height="20" mode="multi"></div>

or as an element:

<chromosome chr="1" rel-size="true" axis="true" assembly="37" width='850' height="20" mode="multi"></chromosome>

multi selector mode

<div chromosome chr="1" rel-size="true" axis="true" assembly="37" width='850' height="20" mode="multi"></div>

single selector mode

<chromosome chr="1" rel-size="true" axis="true" assembly="37" width='850' height="20" mode="single"></chromosome>

no selector mode

<chromosome chr="1" rel-size="true" axis="true" assembly="37" width='850' height="20" mode="none"></chromosome>
