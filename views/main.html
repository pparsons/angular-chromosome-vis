Getting Started

Dependencies

  • AngularJS
  • D3
  • underscore
  • JSDAS

Installation

Install with bower: 

bower install angular-chromosome-vis --save

Declare a dependency on the angular module:

angular.module('myModule', ['angularChromosomeVis']);

Settings

All settings can be provided as attributes on the chromosome directive.

Descriptions

  • chr: The chromosome to represent. Can take a value from 1 to 22 and X or Y.

  • id: id of the element. Is an optional attribute, but is necessary for selector(s) to work properly.

  • rel-size (default: true): Use relative sizing of chromosomes. If false, absolute sizing will be used, which will make all chromosomes the same size (length).

  • axis (default: true): Show axis under chromosome that depicts length in base pairs.

  • assembly (default: 37): Genome assembly to use (e.g., 37, 38).

  • width: Width of chromosome. Can be set to inherit from the parent element, or can be set as a specific number.

  • height (default: 20): Height of chromosome.

  • mode (default: multi): Selection mode. Allow multiple selectors, only one selector, or no selectors.

  • centromere (default: line): How to display centromere location. With line, dot, or none.

Summary

Attribute Optional or Required Possible Values Default
chr Required 1-22, X, Y -
id Optional * -
rel-size Optional true, false true
axis Optional true, false true
assembly Optional 37, 38 37
width Required inherit, [integer] -
height Optional [integer] 20
mode Optional single, multi, none multi
centromere Optional line, dot, none line

Usage and Examples

Add the directive to HTML:

as an attribute:

<div chromosome chr="1" id="chr1" rel-size="true" axis="true" assembly="37" width='850' height="20" mode="multi"></div>

or as an element:

<chromosome chr="1" id="chr1" rel-size="true" axis="true" assembly="37" width='850' height="20" mode="multi"></chromosome>

multi selector mode

<div chromosome chr="1" id="chr1" rel-size="true" axis="true" assembly="37" width='850' height="20" mode="multi" centromere="line"></div>

single selector mode

<chromosome chr="3" id="chr3" rel-size="true" axis="true" assembly="37" width='850' height="20" mode="single" centromere="dot"></chromosome>

no selector mode

<chromosome chr="1" id="chr14" rel-size="true" axis="true" assembly="37" width='850' height="20" mode="none" centromere="line"></chromosome>

inherit width from parent

The width can be set to a specific number, as in the examples above, or it can be set to inherit from the parent element. This can be useful for setting the width with css classes. For example, using bootstrap's grid system:

<div chromosome class="col-sm-12" chr="1" id="chr1" rel-size="true" axis="true" assembly="37" width="inherit" height="20" mode="multi"></div>

<div chromosome class="col-sm-6" chr="1" id="chr1" rel-size="true" axis="true" assembly="37" width="inherit" height="20" mode="multi"></div>
